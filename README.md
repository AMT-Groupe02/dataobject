# dataobject

## Tests
Nous utilisons Junit 5 pour les tests unitaires. Pour lancer les tests, il faut lancer la commande suivante :
```bash
mvn -AWS_ACCESS_KEY_ID={ID} -AWS_SECRET_ACCESS_KEY={SECRET} test
```
